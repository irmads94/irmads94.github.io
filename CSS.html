Reglas

    Permiten cambiar la apariencia(presentación) de los elementos Html.

  • Sintaxis general

    • Selector { Declaraciones }

  • Declaraciones

    • Propiedad : Valor;

Tipos de estilos

  • Por default

    • Son los estilos que ya tiene el navegador definido.

  • Del usuario

    • Son los estilos que se encuentran en un archivo local.

  • Estilo externos

    • Estan definidos en una hoja de estilos externa, utiliza la etiqueta <link> para hacer referencia al archivo.

    Atributos:

    href
    type
    rel

    Ejemplo:
    Dentro del archivo *.css
    h1 { color:blue; }

  • Estilo interno

    • Este tipo de estilo se definen mediante la etiqueta <style>

    Ejemplo:
    Tiene que estar declarado dentro del head.
    <style>
    h1 { color:blue; }
    </style>

  • Estilo en linea

    • Con este tipo de estilo se asigna la propiedad y el valor como atributo del elemento.

    Ejemplo:
    <h1 style="color: blue;">Titulo</h1>

Selectores

  • Selector de tipo

    • Selecciona un tipo de elemento y aplica el estilo a todos los elementos del mismo tipo.

    Ejemplo:
    p { text-align : justify; }
    Todos los parrafos justificados.

  • Selector Descendiente

    • Aplica el estilo a elementos que se estan contenidos dentro de otros elementos.

    Ejemplo:
    p b { color : red; }
    Todo lo que esta en un parrafo en negritas, se pondra en color rojo.

  • Selector hijo inmediato

    • Similar al Descendiente pero mas restrictivo. Aplica el estilo solamente a los elementos que sean hijos directos de otro elemento.

    Ejemplo:
    p > b { color : blue }
    <p><i><b>Hola</b></i></p>
    No es hijo directo, entonces no se aplica el estilo(color azul).

  • Selector por ID

    • Un ID es un atributo de un elemento HTML y se utiliza para aplicar un estilo de manera particular.

    Ejemplo:
    <p id = "introduccion" >Este es un parrafo</p>
    Un selector por ID utiliza el simbolo #
    #introduccion { color : blue; }

  • Selector por clase

    • Aplicar estilo a un conjunto de elementos.

    Ejemplo:
    <p class = "especial" >Este es un parrafo</p>
    .
    .
    .
    <p class = "especial" >Este es otro parrafo</p>
    Un selector por clase utiliza el simbolo .
    .especial { color : red; }

  • Selector universal

    • Aplica el mismo estilo a todos los elementos de la página.
    Los estilos se heredan, si aplicas un estilo a un elemento, todos los elementos dentro de él tendran el mismo estilo, pero se pueden modificar por medio de declaraciones adicionales en cascada.

Agrupación de Elementos

  • DIV

    • Agrupa elementos en bloque.

    <div> ELEMENTOS </div>

  • SPAN

    • Agrupa elementos en linea.

    <span> ELEMENTOS </div>

Propiedades

  • Color

    • Formato permitido:
    Nombre (140)
    Hexadecimal (3 digitos / 6 digitos)
    RGB / RGBA HSL / HSLA

  • Fuente

    • Familia de fuentes

    • Serif

      • Las letras tienen detalles en los extremos. Un claro ejemplo es Times New Roman.

    • Sans-Serif

      • Las letras no tienen detalles pero es de ancho variable. Arial es un tipo de letra Sans-Serif.

    • Monospace

      • Las letras tienen un ancho fijo. Un ejemplo de esta familia es la letra Courier.

    Propiedades de fuentes

    • font-family

      • Tipo de familia(mencionadas arriba) a la que pertenece la fuente.

    • font-style

      • Estilo que puede tener la fuente (Normal, italic, oblique).

    • font-size

      • Tamaño de la fuente definido en px, em y porcentaje(%).

    • font-weight

      • Grosor con el que se muestran las letras del texto(normal, bold).

  • Texto

    • Efectos

      • Permite añadir diferentes "decoraciones" o "efectos" al texto. Por ejemplo, se puede subrayar el texto, tacharlo o ponerle un subrayado superior.

    • Alineación

      • Los valores posibles de esta propiedad son: left(texto alineado a la izquierda), right (texto alineado a la derecha) o center (texto con alineación centrada). Además, el valor justify(alineación justificada) alargará cada línea de forma que los márgenes izquierdo y derecho estén justificados. Esta tipo de presentación la habrás visto, por ejemplo, en periódicos y revistas.

    • Transformación

      • Controla la escritura en mayúsculas de un texto. Puedes elegir entre los valores capitalize, uppercase, olowercase, sin importar cómo aparece el texto original en el código HTML.

Modelo de cajas

  • Acomodo de Cajas

    • Todos los elementos de HTML estan encerradas en una caja.

    Existen dos tipos de acomodo de cajas:

    BLOQUE como por ejemplo <div>, <p>, <img>, <table>

    LINEA como por ejemplo <b>, <i>, <strong>, <em>, <span>

  • Posicionamiento de Cajas

    • Estatico

      • position : default;

      Es el posicionamiento que tiene por default las paginas web. El pos es en flujo y este puede ser en linea o bloques.

    • Absoluto

      • position : absolute;

      Se saca la caja del flujo normal, y se especifica su posición respecto al contenedor.

    • Relativo

      • position : relative;

      Es el posicionamiento que le corresponde de acuerdo al flujo.

    • Fijo

      • position : fixed;

      Se mantiene una posición aun después del scroll. Es muy utilizado en menus.

Margen, relleno y borde

  • Relleno

    • Padding.

    Establece cada uno de los rellenos horizontales y verticales de un elemento.

  • Margen

    • Margin.

    Establece cada uno de los márgenes horizontales y verticales de un elemento.

  • Borde

    • Border.

    Las propiedades de borde de CSS le permiten especificar el estilo y el color del borde del elemento.
