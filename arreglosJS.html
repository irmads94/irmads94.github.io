Creación

Arreglos vacios

var arreglo = new Array();

var arreglo = [];

Arreglos Heterogeneos

var arreglo = Array('hola',123,true);

var arreglo = ['mundo',true,1];




Arreglos Bidimensionales

var matriz = Array(Array(1,2,3),Array(4,5,6));

var matriz = [[1,2,3],[4,5,6]];




Arreglos Jagged(Dentados)









var jagged = Array(Array('A','B'),Array('C','D','E','F'),Array('G'));

var jagged = [['A','B'],['C','D','E','F'],['G']];









Acceso

Teniendo var arreglo = new Array('mundo','hola',123,true);

y enseguida inserto un numero(456) en la posición 10 (Sabiendo que solo tengo 4 elementos), JavaScript no marca error o excepción, si no que lo inserta en la posición 10, y de la posición 4 a la 9 queda un tipo de dato "indefinido(undefined)", quedando de la siguiente manera:

Array('mundo','hola',123,true,undefined,undefined,undefined,undefined,undefined,undefined,456)

En otros lenguajes de programación, intentar acceder a un índice de array inexistente devuelve un error, pero en JavaScript no es así. Si escribimos arreglo[40] cuando sólo hemos definido hasta el índice 10, el resultado es que arreglo[40] devuelve undefined. A diferencia de en otros lenguajes, los arrays en JavaScript no tienen un número fijo de elementos, sino que el número de elementos del array se ajusta dinámicamente según las necesidades.

Métodos

  • concat() - Concatena 2 arreglos, regresa un arreglo nuevo.
  • join() - Une todos los elementos de un arreglo, regresa una cadena.
  • push() - Agrega un elemento al final del arreglo.
  • pop() - Elimina el ultimo elemento del arreglo.
  • reverse() - Regresa un arreglo con los elementos en orden invertido.
  • sort() - Ordena los elementos de manera ascendente (abecedario).
  • shift() - Elimina al inicio
  • unshift() - Inserta al inicio
  • splice() - Agregar/eliminar elementos en cualquier posición.
  • slice() - Extrae un subarreglo.
  • indexof() - Busca un elemento en el arreglo, regresa el indice, (-1, en caso de que no exista)
  • foreach(arreglo, elemento,indice) - Iteraciones en un arreglo, mediante una función

Funciones

Declaración

                        function nombreFuncion(parametros)

                            }

Los parametros pueden ser opcionales

Expresión

                          var nombreFuncion = function nombreOpcional(parametros)
                                                {
                                                    instrucciones;
                                                }
NOTAS

Los parametros pueden ser opcionales

En caso de no poner nombre a la función se le denomina "Función anonima"
