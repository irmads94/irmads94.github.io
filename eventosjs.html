Funcionamiento

elemento → evento → código

           asocia    ↓       ejecutar

Tipos de eventos

Interfaz de usuario
  • scroll
  • resize
  • load
Teclado
  • key press
  • key down
  • key up
Ratón
  • click
  • double click
  • mouse down
Formularios
  • input
  • submit
  • change

Pasos para el manejo de eventos

1.

Seleccionar el elemento(nodo)

2.

Indicar el evento en el elemento seleccionado

3.

Escribir el código(script) a ejecutar en respuesta al evento.

Tipos de manejadores de eventos

Manejador HTML

Sintaxis:

<elemento> onEvento = "código"> Contenido </elemento>

Ejemplo:

<a> onclick = "alert('Hola')"> Saluda </a>

Manejador DOM Tradicional

Desventaja: 1 evento a 1 elemento.

Sintaxis:

elemento.onevento = nombreFunción

Ejemplo:

document.getElementById('enlace').onclick = function(){ alert ('Hola')}

Escuchador DOM nivel 2

Sintaxis:

elemento.addEventListener ('evento',nombreFunción)

Ejemplo:

document.getElementById('enlace').addEventListener('click',saludar)
