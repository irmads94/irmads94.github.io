Conceptos Básicos

HTML

Hypertext Markup Languaje

Las páginas HTML son documentos de texto

HTML usa elementos para definir la estructura de la página

Elementos

Los elementos se definen por etiquetas.

Contenido

El Contenido de un elemento se encierra entre etiquetas

Etiquetas

Las etiquetas tienen que estar encerradas en parentesis angulares < y >

Existen etiquetas de apertura <etiqueta> y de cierre </etiqueta>

El nombre de la etiqueta debe ser el mismo y se distingue a la etiqueta de cierre con una diagonal

<etiqueta> CONTENIDO </etiqueta>

Pueden tener Atributos los cuales se especifican en la etiqueta de inicio

Atributos

Contiene información adicional acerca del Contenido

<etiqueta atributo = "valor" > CONTENIDO </etiqueta>

Parrafos

Muchas veces, los encabezados tienen el propósito de enfatizar o agrandar solo unas pocas palabras. Si desea agregar el contenido en formato de párrafo, puede añadir un párrafo mediante el elemento de párrafo <p>.

Los párrafos son ideales para la expansión de la cantidad de contenido (texto) en una página web. Se tiene que tomar en cuenta que grandes cantidades de texto en formato de párrafo puede abrumar los visitantes de la página web.

Por ejemplo, si varios párrafos en su página web contienen grandes cantidades de texto, la página web podría llegar a ser difícil de consumir.





Saltos de linea

Para poder hacer un salto de linea en HTML se utiliza el elemento <br/> El elemento de salto de línea es una etiqueta de cierre automático. Se puede utilizar en cualquier lugar dentro del código HTML y un salto de línea se mostrará en el navegador.

Ejemplo:

Este es un parrafo <br> Y este otro.

El código en el ejemplo anterior dará lugar a una salida que tiene el siguiente aspecto:

Este es un párrafo

Y este otro.

Nota : Los saltos de línea no son la forma habitual de manipulación de la colocación de los elementos HTML.

Carácteres especiales

Los motivos por los que debemos usar los caracteres especiales son los siguientes:

Una página web se ha de ver en países distintos, que usan conjuntos de caracteres distintos. El lenguaje HTML nos ofrece un mecanismo por el que podemos estar seguros que una serie de caracteres "raros" se van a ver bien en todos los ordenadores del mundo, independientemente de su juego de caracteres local.

Por otra parte, ciertos caracteres forman parte de las etiquetas como el "mayor qué" o "menor qué" que las delimitan. Mediante los caracteres especiales se definen unos códigos, que debemos usar cuando queremos poner uno de estos caracteres en una página.

Estructura semántica

Encabezado

<header>

Es el equivalente a la cabecera de la página web. Contiene el título o nombre de la empresa/titular de la página, logo e información relacionada.

Navegación

<nav>

Contiene los enlaces (barra de navegación) externos o internos de la página.

Contenido Principal

<main>

Utilizaremos la etiqueta para abarcar la parte de la web donde se encuentra el contenido que los visitantes quieren leer.

Sección

<section>

Es una gran caja que sirve para mostrar grandes bloques de contenido de la página. Puede contener diferentes subapartados de diferentes temas (de tipo <article>).

Artículo

<article>

Es una caja independiente de contenido que puede estar contenida (o no) dentro de un <section>. Normalmente utilizada para contenidos no demasiado extensos.

Barra Lateral

<aside>

Define un bloque de contenido relacionado de manera indirecta con el contenido principal, pero que no es esencial para la compresión del mismo.

Pie de página

<footer>

Equivale al pie de página de un apartado concreto (<section>, <article>...) o de la página web en general.

Imágenes

La etiqueta <img> que funciona para colocar imágenes dentro de una página web. Para poner una imágen utilizamos el siguiente código:

<img src="imagenHTML.jpg">

Donde "imagenHTML.jpg" es la imagen que vamos a colocar. Deberá estar ubicada en el mismo directorio. Si queremos colocar una imagen con HTML en una página que no está colocada dentro del mismo directorio, podemos usar una ruta absoluta:

<img src="folder/imagenHTML.jpg">

  • ETIQUETA

    • <img>

  • ATRIBUTOS

    • src="Nombre del archivo"
    height="Altura"
    width="Ancho"
    alt="Texto alternativo(¿De que trata esa imagen?)"

Tipos de Listas

  • Listas Ordenadas

    • Si lo que pretendemos es definir una lista ordenada, lo tendremos que hacer entre las etiquetas <ol> (ol indica ordered list)

    Cada elemento de la lista se escribirá con la etiqueta <li>.Al ser listas ordenadas los símbolos por defecto serán números y éstos se irán generando automáticamente por orden, conforme escribamos nuevos elementos de la lista.

  • Listas No Ordenadas

    • Las listas no ordenadas son aquellas que se encuentran entre las etiquetas <ul> (ul indica unordered list)

    Cada elemento de la lista se escribirá con la etiqueta <li>.










  • Listas de Descripción

    • Las listas de definiciones se usan cuando queremos hacer una enumeración tipo “diccionario” donde tenemos varios términos y su definición o descripción.

    Estos términos y sus definiciones o descripciones podríamos ponerlos de varias maneras dentro de una página web (como texto sin más, como lista ordenada, no ordenada…) y una de estas maneras es ponerlo como lista de definiciones.

    Para crear una lista de definiciones debemos usar las etiquetas:

    <dl>
    Indica que dentro de ella va a ir una lista de definiciones o lista de descripciones.

    <dt>
    Indica que dentro de ella va un término que vamos a definir.

    <dd>
    Dice que dentro de ella se encuentra una definición o descripción asociada a un término. Un término podría tener varias descripciones. Por ejemplo el término Autor podría tener como descripciones: Mateo Renzi, Olivo Pascua, Jorge Guillén.

Tablas

  • Tabla

    • <table>

    Se emplea para definir tablas de datos

  • Descripción

    • <caption>

    Se emplea para definir la leyenda o título de una tabla

  • Encabezado

    • <thead>

    Se emplea para definir los títulos de la tabla. Tenga en cuenta que los encabezados de la tabla aún requiere de una fila para contenerlos.

  • Pie de página

    • <tfoot>

    Se emplea para contener los totales de los datos de la tabla. Los pies de página se utilizan a menudo para contener sumas, diferencias, y otros resultados de los datos.

  • Cuerpo de la página

    • <tbody>

    Se emplea para contener todos los datos de la tabla, con exclusión de los encabezados de la tabla

  • Fila

    • <tr>

    Se emplea para definir cada fila de las tablas de datos

  • Celda del encabezado

    • <th>

    Se emplea para definir las celdas que son cabecera de una fila o de una columna de la tabla

  • Celda del cuerpo y pie de página

    • <td>

    Se emplea para definir cada una de las celdas que forman las filas de una tabla, es decir, las columnas de la tabla
