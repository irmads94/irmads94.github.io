Introducción

    "ECMAScript" lenguaje estandar JavaScript

    Surge por Brendan Eich a traves de la empresa Netscape en 1995, en ese entonces no era "JavaScript" era Mocha y LiveScript, este ultimo desarrollado por Microsoft.

    Java fue creado por Sun Microsystems(Oracle) en 1993, mientras que la implementación de JavaScript fue creado por Netscape y Microsystems.

    Es un lenguaje interpretado, y por lo tanto no se compila.

    Basado en objetos y en DOM(Document Object Model)

Uso y aplicaciones

  • Animaciones
  • Geolocalización
  • Canvas
  • Audio y Video
  • Procesamiento de Formularios
  • Ajax -Comunicación servidor- comunicación asíncrona
  • Elementos HTML dinámicos
  • Juegos en el navegador
  • Efectos y elementos flotantes
  • Programación del lado del servidor -Node.js-

Objetos, propiedades y métodos

    Lenguaje

  • Operadores aritméticos

    • Suma( + ), resta( - ), multiplicación( * ), división( / ), modulo( % ), incremento en uno( ++ ), decremento en uno( -- ), entre otros.

  • Operadores relacionales

    • Mayor que( > ), menor que( < ), mayor o igual( >= ), menor o igual( <= ), diferente( != ), igual en valor( == ), igual en valor y en tipo( === )

  • Operadores lógicos

    • And( && ), or( || ), not( ! )

  • Condicionales

    • If, if-else, switch/case/break/default

  • Ciclos

    • While, do-while, for, break, continue

  • Tipos de datos

    • Cadena, número(entero y decimal), booleano, arreglos, funciones

    Cast: Number(), string(), boolean(),parseFloat(), parseInt()

  • Clases de soporte

    • Math, date

  • NOTA: SI NO SE PONE ";" SE TOMA COMO SEPARADOR DE INSTRUCCIONES EL SALTO DE LINEA

Objetos de salida

OBJETO MÉTODOS DESCRIPCIÓN
