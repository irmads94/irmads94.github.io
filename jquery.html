jQuery()

Todo se realiza a traves de una función JQuery()

Recibe como parámetro una expresión CSS y regresa un elemento JQuery

Ejemplo:

var objeto = jQuery(SelectorCSS)

objeto es un "Objeto jQuery"

Tipos de selectores

*

elemento

#id

.clase

sel1,sel2

Ejemplo:

var todos = jQuery('*')

Filtros básicos

:first - Regresa el primero

:last - Regresa el ultimo

:even - Regresa objetos en posición par

:odd - Regresa objetos en posición impar

:eq(index) - Regresa el objeto que esta en la posición(index)

:gt(index) - Regresa objetos mas alla de la posición(index)

:lt(index) - Regresa el objeto que esta antes de una posición(index)

:not(selector) - Negación

Ejemplo:

var primero = jQuery('*:first)

var segundo = jQuery('*:eq(1)')

var sinParrafos = jQuery('*:not(p)')
