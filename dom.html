Introducción

Document Object Model

Representa un documento HTML en forma de una estructura de árbol.

El documento se puede manipular facilmente a traves del árbol, es decir, agregar, eliminar, modificar elementos, etc.

Acceder elementos del documento

Métodos del documento

document.getElementsByTagName('etiqueta') - regresa un arreglo de Nodos.

document.getElementsByClassName('clase') - regresa un arreglo de Nodos.

document.getElementById('identificador') - regresa un nodo.

Modificar el contenido de un elemento

Atributos

elemento.textContent = 'texto'

elemento.innerHTML = 'código HTML'

Agregar elementos(nodos) nuevos

5 Pasos

  • Crear el elemento - createElement()
  • Crear el contenido - createTextNode()
  • Añadir el contenido al elemento - appendChild()
  • Agregar atributos - setAttribute()
  • Agregar el nuevo elemento al documento - appendChild()
